Algos Package
=============

This package supports the actual algorithmic workload. It is divided into distinct groups or themes that are prevalent
in the algorithmic literature. Some modules, such as :mod:`algos.io`, are convenience utilities to support the
algorithmic processing.

+--------------------------------------+---------------------------------------------------------------------------+
| module                               | purpose                                                                   |
+======================================+===========================================================================+
| :mod:`algos.io`                      | Supports a variety of input/output methods e.g. reading from ``stdin``.   |
+--------------------------------------+---------------------------------------------------------------------------+
| :mod:`algos.text`                    | Algorithms that operate on textual input                                  |
+--------------------------------------+---------------------------------------------------------------------------+
